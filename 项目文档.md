## 2. 设计风格

### 2.1 整体风格
- **设计语言**：Apple极简风格 + Eva式审美
- **背景色**：`#FBFBFD`（浅灰白）

- **大圆角**：`28px - 32px`

## 3. 技术架构

### 3.1 技术栈
```
前端框架：纯静态HTML
样式框架：Tailwind CSS (CDN)
图标库：Lucide Icons
字体：Google Fonts (Inter + Fira Code)
交互：原生JavaScript
```

### 3.2 技术决策

#### ✅ 纯前端方案（已采用）
**优点**：
- 部署简单，零维护成本
- 加载速度快，SEO友好
- 无需服务器，支持GitHub Pages/Vercel
- 符合项目需求（静态内容为主）

**缺点**：
- 内容更新需要重新部署

#### ❌ 后端方案（未采用）
不需要，因为：
- 无用户登录/数据提交
- 无动态数据查询
- 内容不频繁更新

#### ✅ 数据存储方案
**当前方案**：内容直接写在HTML/配置文件

**优点**：
- 简单直接
- 版本控制友好
- 无需数据库

**未来可选升级**：
- Headless CMS（Contentful/Notion API）
- JSON/YAML配置文件 + 构建时注入

## 5. 已解决的问题

**问题**：原始demo的3个核心能力（平台运营、文案策划、数据分析）内容不够详细

    background: #FFFFFF;
